# 剧情

# SP1：

恢复意识的她，发现自己苏醒于这个梦幻的地方。

“多么令人愉快啊，”她想，她蹲了下来，整了整白色的裙子，环顾四周才发现这附近竟然有绿蓝色的宝石。

那些事物——不依靠任何外力便飞舞于空中。

“太美妙了！”她自心底赞叹道。这些宝石反射出了另一个纯洁的世界。她从中看见海洋、城市、火焰、光芒；美好的景象目不暇接。她抬起了自己的手，试图去抓住它们，开心地笑了出来。

她并不知道这些宝石有个名字：神迹石。

实际上，名字对这些过于美好的事物来说并不重要。她触碰、旋转、观察它们；她靠这样来娱乐自己。这已经足够了，难道不是吗？现在留给她的，有三个问题，我是谁，我在哪，我要去哪里。在这些疑问的包围下，她没有问出任何一个，也不想得到任何答案。神迹石的光芒已经使她心满意足。这是她与这个新世界的邂逅······

# SP2：

黑衣少女醒来在一座损毁的塔楼中。

飘浮着的蓝紫色碎片是她第一个注意到的事物。

它们引领她前往了室外——那五彩斑斓的世界。

它们看上去正在被她吸引而来，而被激起了好奇心的她开始观察这些蓝紫色碎片——陨星石。

就像透过火车车窗看外头稍纵即逝的景色一般，她瞧见了阴雨的景象。下一次是艳阳。再下一次是死亡。她厌恶地远离了这些碎片。

虽然总是紧随着她，可这些碎片总能在少女试图捏碎它们时躲开。

少女心中的厌恶渐渐地化为为愤怒，而她迫使自己把注意力转移到那灰白色的天空。然而，在她仰望天空之际，原先脸上的情绪荡然无存。

她的嘴微微张开来，却因过于惊讶而说不出半句话来碎片在高空中搅动着、闪烁着、旋转着。

这看着就像是场玻璃碎片的暴风雨。她后悔把注意力转移到天空上。

但碎片们已经发现了她，渐渐降落下来，要与她打个招呼······
