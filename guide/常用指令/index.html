# 指令帮助

# 传送指令

/back

返回上一位置tpa 请求传送他人

/tpahere

请求他人传送到自己位置-

/sethome

家名字 当前位置设置家

/home 家名字

传送回家（上限：3)

# 经济指令

/balance

查询自己兔兔币

/baltop

查看TOO经济榜单

pay 玩家名字 数额

向他人转账

/paytoggle

开启/关闭接收转账

# 动作指令

/sit

坐下(shift键坐下后起立)

/hat

戴上手中物品

shift键+右键

部分物品/武器触发交互

# 查询指令

/co i

开启查询模式 开启查询模式后，左键物品查找记录·记录包括且不限于放置，破坏，红石·对于无法查询到的记录,请联系管理 (再次输入co i后关闭查询模式)

/co near

查询身边5格内所有的方块记录

# 箱子商店

/qs create

将目标箱子设置商店使用物品左键目标箱子快捷建立商店 右键商店箱子告示牌快捷编辑商店

/qs remove

移除目标商店

/qs buy

改为收购商店

/qs sell

改为出售商店

/qs price

更改商店物品价格

# 领地帮助

左键打开领地权限面板(在目标领地内）

/res create

创建领地 在输入指令前先用木锄 左右键分别选取创建区域对角

/res remove

删除领地

/res area

查看选中区域价格

/res tp 领地名

传送至领地

/res message 领地名 enter/leave

更改进入/离开领地显示信息

/res padd 玩家名

给予玩家领地权限

/res tpset

设置领地传送点

/res expand 格数

向面向方向扩展领地

# 皮肤帮助

/skin set

正版ID设置为ID的皮肤

/skin clear

清除身上设置的皮肤

/skin url 图床直链 classic/silm

将你想自定义的皮肤上传至图床网站，获取直链 再使用该指令穿戴Steve/Alex模型皮肤如果有不明白的地方，欢迎询问管理腐竹来帮你解决哦
