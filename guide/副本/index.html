# 副本*古遗迹战场

·副本/组队

·副本仅为拓展玩法,并非专业RPG

·诸多问题,还望包容,及时反馈,不甚感激

# 副本须知

组队后只需要队长开启副本开启副本需要小队集体在线

有成员下线后,可以删除重建小队

或将其移除建议离开副本及时删除小队由于高版本原因,插件不稳定

由于等待版本更新,组队暂未内核汉化遇见问题,请及时反馈,给予补偿

为启用旁观模式,需要手动输入"结束"离开副本注意!输入结束后,所有人会离开副本!

禁用物品(若有成员携带会无法进入副本):不死图腾,跳跃药水,鞘翅,末影珍珠
