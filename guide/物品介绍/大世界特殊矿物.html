# 陨星矿：

在神与魔王大战时由于魔王击碎星辰，少量破碎的星辰碎片坠落到了大地，非常的稀少，如今世上只有地表深层才能找到了。

# 陨星矿石：

生成在地下0~16层

分布在所有群系，有三种矿脉类型，生成概率根据矿脉所含矿石数量递减

受时运效果影响

硬度：5

可挖掘工具：铁镐及以上镐

# 深层陨星矿石：

生成在地下0~-64层

分布在所有群系，有三种矿脉类型，生成概率根据矿脉所含矿石数量递减，

稀有度：平均每区块生成3.27

不受时运效果影响

硬度：8

可挖掘工具：铁镐及以上

# 神迹石：

在神与魔王大战时，受神力影响产生的矿物，非常的稀少，如今世上只有地表深层才能找到了。

# 神迹石矿

生成在地下0~-64层

分布在所有群系，有三种矿脉类型，生成概率根据矿脉所含矿石数量递减

稀有度：平均每区块含1.04

不受时运效果影响

硬度：10

可挖掘工具：下界合金镐
